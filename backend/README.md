# SkinLesion AI – Backend API

FastAPI backend cho hệ thống chẩn đoán da liễu.

## Endpoints

- `POST /api/diagnose` — Phân tích ảnh tổn thương da
- `POST /api/chat` — Chatbot tư vấn da liễu  
- `POST /api/disease-info` — Thông tin chi tiết về bệnh
- `GET /health` — Health check
